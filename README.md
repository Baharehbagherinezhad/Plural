Plural – پروژه ساده وب
--------------------------------------------------------------------------------------------------------------------------------------------------------------

Plural یک پروژه کوچک HTML/CSS/JavaScript است که یک رابط کاربری ساده با طراحی و کمی تعامل (interactivity) را نمایش می‌دهد.

این پروژه به عنوان تمرین یادگیری عملی توسعه وب ساخته شده و می‌تواند نمونه‌ای از مهارت‌های پایه Frontend باشد.

 ویژگی‌ها

✔ ساختاردهی صفحه با HTML
✔ طراحی واکنش‌گرا و استایل‌دهی با CSS
✔ تعامل اولیه با کاربر با استفاده از JavaScript
✔ نمایش نحوه کار با DOM و رویدادها

 عملکرد پروژه

این پروژه یک صفحه وب تعاملی ساده ایجاد می‌کند که کاربر می‌تواند عناصر آن را مشاهده و با آن‌ها تعامل داشته باشد.
برای اجرای پروژه کافی است فایل HTML را در مرورگر باز کنید و عناصر CSS و JavaScript آن را مشاهده کنید.
